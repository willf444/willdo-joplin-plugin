# WillDo

WillDo is a task manager plugin for Joplin focused on speed and practical use.

## Features

- Create tasks quickly with Enter
- Edit title and date/time
- Native alarm support through Joplin to-do mirrors
- Daily, weekly, monthly, and yearly recurrence
- Search with visual highlight
- Link a task to the currently open Joplin note
- Open the linked note directly from the task
- Separate pending and completed tasks

## Typical workflow

1. Create the task quickly
2. Set date/time when needed
3. Link it to the relevant Joplin note
4. Let Joplin handle the notification
5. Open the linked note directly from the task

## Build

Run: npm run dist

The generated .jpl file will be created in the publish/ folder.
